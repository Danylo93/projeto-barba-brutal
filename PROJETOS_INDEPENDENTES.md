# 🚀 **PROJETOS INDEPENDENTES - SAAS COMPLETO**

## 📁 **ESTRUTURA SEPARADA:**

```
projeto-barba-brutal/
├── backend/          (NestJS - API)
├── frontend/         (Next.js - Web)
├── mobile/           (React Native - Mobile)
└── README.md
```

## 🔧 **EXECUÇÃO INDEPENDENTE:**

### **1. BACKEND (NestJS):**
```bash
cd backend
npm install
npm run dev
# http://localhost:3001
```

### **2. FRONTEND (Next.js):**
```bash
cd frontend
npm install
npm run dev
# http://localhost:3000
```

### **3. MOBILE (React Native):**
```bash
cd mobile
npm install
npm start
# Expo dev server
```
